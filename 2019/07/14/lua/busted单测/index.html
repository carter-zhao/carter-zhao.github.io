busted入门指南

centos上安装

安装luarocks

yum 安装openresty、openresty-resty

也可以源码安装

安装busted

luarocks install busted

luarocks list 看到安装的busted 是2.0.rc13-0 版本 ， 默认busted 程序在/usr/bin下

安装 lua-resty-busted

  • luarocks install lua-resty-busted

默认在/usr/local/bin/busted , 注意这和 /usr/bin/busted 不一样

  • 设置 LUA_PATH 和LUA_CPATH

执行 luarocks path，将产生的2条export 执行即可

  • /usr/local/bin/busted –version 会报错

luarocks list 看到busted版本变成了2.0.rc11-0， 之前是2.0.rc13-0，这个报错是2.0.rc11-0版的一个bug，而lua-resty-busted 的依赖busted版本是2.0.rc11-0，所以在安装lua-resty-busted 将之前busted 2.0.rc13-0 删除，替换成了2.0.rc11-0，需要替换回来

再次/usr/local/bin/busted –version ， ok
